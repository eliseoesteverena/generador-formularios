Markdown to Form Generator

Syntax Guide:

  • Use # Title for main title
  • Use ## Section for sections
  • Questions are automatically detected if they end with ?
  • Add descriptions with @desc: Your description text on the line after a question
  • Response types:
    • @short - Short text input (default)
    • @long - Long text input (textarea)
    • @multi - Multiple choice (checkboxes)
    • @single - Single choice (radio buttons)
    • @file - File upload
